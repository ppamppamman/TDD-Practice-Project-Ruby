## 도서 대출 시스템
---
도서 대출 케이스 (큰 틀)
```
1. 책을 검색한다
2. 책의 대여상태를 확인한다
3. 대여중이 아니면 대여할 수 있다
4. 대여중이면 예약할 수 있다
5. 책을 받는다
```
---
1. 검색
```
- 책 제목 / 작가 / 출판명으로 책을 검색한다
- 책이 있으면 책의 정보를 보여준다
- 책이 없으면 에러를 발생시킨다
```

2. 대여
```
- 책이 대여 중인지 확인한다
- 대여중이 아니면 책을 대여한다
- 대여중이면 예약한다
```

3. 반납
```
- 대여한 책은 5일간 볼 수 있다
- 5일이 지나면 자동으로 반납된다
```

4. 반납 연장
```
- 대여한 책은 반납일 연장이 가능하다
- 예약이 있으면 반납일 연장이 불가능하다
```

5. 예약
```
- 대여중인 책은 예약할 수 있다
- 예약은 5명까지 가능하다
```
